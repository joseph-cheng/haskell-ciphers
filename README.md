# Haskell Ciphers

This is a set of tools used to encrypt, decrypt, and crack ciphers. All written in Haskell.

## Dependencies

```
Haskell Platform
Haskell 'random-shuffle' package
```

## Usage

To hillclimb:
```
ghc -main-is HillClimbing.main --make -O
```
Windows:
```
HillClimb.exe
```
Linux:
```
./HillClimb
```
